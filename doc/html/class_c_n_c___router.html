Detailed Description

The class controls the motors of the xy router of the CNC.

The class manages the motion control of the XY router.

Author
Francesco Giurlanda

void CNC_Router::searchHomePos ( )

It moves the utensil up to the home position. It is identified by the down limit switches on the YX-axis.

Pay attention to check the correct installation of the down limit switches. If they do not work well, the function can damage your machine.

void CNC_Router::setAbsolPos ( )

void CNC_Router::setIncrPos ( )

It sets up a high precision mode to control the XY-axii.

The function sets up the EIGHTH_STEP for A4988 and HALF_STEP for ULN2003A.

See also
MSMC_A4988.h MSMC_ULN2003A.h

It sets up a low precision mode to control the XY-axii. It is also the fastest control mode.

void CNC_Router::setMotionModeX ( uint8_t  m)

Specifies the motion control mode for the motor on X-axis.

Parameters
mThe motion mode can be:
  • with ULN2003A controller
    1. FULL_STEP
    2. HALF_STEP
  • with A4988 controller
    1. FULL_STEP
    2. HALF_STEP
    3. QUARTER_STEP
    4. EIGHTH_STEP
    5. SIXTEENTH_STEP
